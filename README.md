# 🚍 Bus Simulator Lubumbashi 3D

Un jeu de course 3D amusant où vous pilotez un bus à travers la ville de Lubumbashi !

## 🎮 Jouer en ligne

**Accédez au jeu ici :** [https://dankongolo212-lgtm.github.io/bus_simulator_lubumbashi_3D/](https://dankongolo212-lgtm.github.io/bus_simulator_lubumbashi_3D/)

## 🕹️ Commandes

- **Flèches Gauche/Droite** - Déplacer le bus
- **Tactile** - Sur mobile, touchez les côtés de l'écran

## ✨ Caractéristiques

- 🚍 Bus 3D détaillé avec textures
- 🏙️ Environnement urbain dynamique avec bâtiments et arbres
- 🚗 Obstacles (voitures) à éviter
- 💰 Collecte de pièces pour des points
- 📊 Système de niveaux progressifs
- 💾 Sauvegarde du meilleur score en local
- ⚡ Optimisation de la fluidité
- 🎨 Éclairage réaliste avec ombres

## 🛠️ Technologie

- **Three.js** - Rendu 3D Web GL
- **JavaScript** - Logique du jeu
- **HTML5 & CSS3** - Interface et style

## 📁 Structure

- `index.html` - Fichier principal
- `game.js` - Logique du jeu et scena 3D
- `style.css` - Styles et UI
- `scene.gltf` - Modèle 3D du bus (format GLTF)
- `textures/` - Textures pour le modèle

## 🚀 Installation locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/dankongolo212-lgtm/bus_simulator_lubumbashi_3D.git
   ```

2. Ouvrez `index.html` dans votre navigateur

Ou utilisez **Live Server** dans VS Code :
- Installez l'extension "Live Server"
- Cliquez droit sur `index.html`
- Sélectionnez "Open with Live Server"

## 🎯 Gameplay

- Collectez les pièces pour augmenter votre score
- Évitez les voitures qui circulent
- Le jeu devient plus difficile au fur et à mesure
- Essayez de battre votre meilleur score !
